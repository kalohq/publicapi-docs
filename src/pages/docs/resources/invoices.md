# Fields

Invoices are created either automatically or on-demand as a result of a freelancer completing a task. The Invoice resource currently exposes the following fields:
- id
- amount
- currency
- status
- reference
- vat
- created_at
- team
- freelancer
- task

# Endpoints

## GET /invoices/

List all the available invoices.
